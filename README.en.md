## INSTALLATION 

### Create a virtual environment with virtualenv

```bash
cd repcompbio_website/
virtualenv .venv
source .venv/bin/activate
```

### Install dependencies

```bash
cd repcompbio_website/
pip3 install -r requirements.txt
```
