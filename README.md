# Website Reprodutibilidade Computacional Bio

## INSTALAÇÃO

### Criação de ambiente com virtualenv

```bash
cd repcompbio_website/
virtualenv .venv
source .venv/bin/activate
```

### Dependências

```bash
cd repcompbio_website/
pip3 install -r requirements.txt
```
