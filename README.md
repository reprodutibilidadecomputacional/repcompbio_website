## INSTALAÇÃO

* Estaremos utilizando a versão de Python 3.9

### Criação de ambiente com virtualenv

```bash
cd repcompbio_website/
virtualenv --python=python3.9 .venv
source .venv/bin/activate
```

### Dependências

```bash
cd repcompbio_website/
pip3 install -r requirements.txt
```
